# 📚 Book Store Management System

A full-stack Spring Boot web application for managing books.

---

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- MySQL
- HTML, CSS

---

## ✨ Features
- Add new books
- View book list
- Edit & delete books
- My Book List
- Clean UI using Thymeleaf
