# 📚 Book Store Management System

A full-stack Spring Boot web application for managing books.

---

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring MVC
- Hibernate
- Thymeleaf
- MySQL
- HTML, CSS

---

## ✨ Features
- Add new books  
- View book list  
- Edit & delete books  
- My Book List  
- Clean UI using Thymeleaf  

## 💡 About This Project
This project was built with the help of tutorials and self-learning.
