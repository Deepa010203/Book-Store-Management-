📚 Book Store Management System

A full-stack Spring Boot web application for managing books with a clean and simple UI.

#🚀 Tech Stack
☕ Java
🌱 Spring Boot
🧩 Spring MVC
🗄️ Hibernate / JPA
🐬 MySQL
🎨 Thymeleaf
🌐 HTML, CSS


#✨ Features
➕ Add new books
📖 View all books
✏️ Edit book details
❌ Delete books
📚 My Book List
🎯 Clean and responsive UI using Thymeleaf
